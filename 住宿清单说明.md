# 共享住宿清单

网站从 `accommodations.json` 读取住宿信息。每次在 GitHub 网页中编辑并提交该文件后，所有人刷新行程页面即可看到最新清单。

## 如何修改

1. 用被邀请为此仓库协作者的 GitHub 账号打开网站“住宿记录”中的“在 GitHub 中编辑清单”。
2. 在 `accommodations.json` 中新增、修改或删除一项记录。
3. 点击 **Commit changes** 提交；可在提交说明中写明修改内容。
4. 回到行程网页刷新即可查看更新。

每条记录格式如下；各字段均使用英文双引号，除价格外均为文本：

```json
{
  "id": "enshi-20260726",
  "stayDate": "2026-07-26",
  "name": "恩施女儿城附近酒店",
  "price": 680,
  "status": "booked",
  "address": "酒店地址、停车入口或导航提示",
  "deadline": "2026-07-24"
}
```

`status` 只能填写 `pending`（待决定）或 `booked`（已预订）。多条记录之间以逗号分隔，最外层始终保留方括号。
